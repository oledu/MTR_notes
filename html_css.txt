CSS cascading style sheet
1.在行內使用ＣＳＳ
 <p style="color:red; font-size:20px; ...>abc</p>

2.放在head裡面
<head>
	<style>
		h1{...}
		.class_name{...}
		#id_name{...}
		順序 id > class > tag
		#id_name tag. //tag in id
		#id_name > tag //id底下第一層的tag
	</style>
</head>

3.分成兩個檔案
<head>
	<link rel="stylesheet" type="text/css" href="css_filename.css">
</head>
